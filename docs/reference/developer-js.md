# 开发者 JS 沙箱

Twilight 提供受控 JavaScript 沙箱，用于扩展 Telegram Bot 指令。该能力面向管理员，不面向普通用户。

## 运行模型

- 运行时为 Goja。
- 单次执行同步且有超时限制。
- 不暴露文件系统、进程、模块加载器、原始数据库、密钥、Token、密码或 API Key。
- 预览模式不应产生持久写入。

## 文档入口

- 管理员页面：`/admin/developer/js-docs`
- 后端接口：`GET /api/v1/admin/developer/js-docs`

## 推荐流程

1. 解析输入参数。
2. 使用断言或显式退出做参数校验。
3. 读取受控上下文或用户状态。
4. 执行小范围、可审计的动作。
5. 返回简短结果。

## 维护要求

新增沙箱 API 时，需要同步后端结构化文档和前端文档面板。用户可见说明应通过 i18n 或中文文档维护。
