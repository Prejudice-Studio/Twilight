# 前端 i18n 指南

Twilight WebUI 使用轻量级语言目录，文件位于 `webui/src/locales`。

## 语言文件

- `basic.json`：简体中文基底和默认回退。
- `zh-Hant.json`：繁体中文。
- `en-US.json`：英文。
- `zh-Hans.json`：兼容占位文件，保持稀疏，不在这里新增常规文案。

## 新增文案流程

1. 在 `basic.json` 添加键。
2. 在 `zh-Hant.json` 和 `en-US.json` 添加对应翻译。
3. 在组件中使用 `useI18n().t()`。
4. 检查英文和繁体中文较长时布局是否稳定。

## 布局要求

- 语言切换器、下拉框、按钮和统计卡片要有稳定尺寸。
- 紧凑控件优先使用图标与 tooltip。
- 不要在已接入 i18n 的页面硬编码中文或英文。
- 提交前扫描语言文件，确保没有连续问号或 mojibake。
