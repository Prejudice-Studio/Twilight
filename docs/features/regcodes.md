# 注册码与卡码

卡码系统用于注册、续期、白名单和诱饵码等场景。

## 类型

- 注册码：用于创建或授权新账号。
- 续期码：用于延长账号到期时间。
- 白名单码：按配置授予白名单角色。
- 邀请来源续期码：由邀请系统为已有下级生成。

## Source 字段

`RegCode.Source` 用于区分来源：

- `admin`：管理员后台生成。
- `invite`：邀请系统生成。
- 空字符串：历史数据，按 `admin` 处理。

## 有效期暂停

注册码停用期间暂停有效期倒计时，依赖 `PausedSeconds` 和 `PauseStart`。使用次数耗尽的优先级高于手动停用。

## 批量生成

批量生成应继续复用 `UpsertRegCodes`，避免回退到逐条写入。
