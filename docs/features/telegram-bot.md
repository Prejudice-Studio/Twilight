# Telegram Bot

Telegram Bot 提供账号绑定、自助命令、管理员命令、通知、花名册和开发者 JS 扩展能力。

## 用户能力

- 绑定或解绑 Telegram 账号。
- 接收登录通知和工单通知。
- 使用已配置的自助命令查看账号状态或重置密码。

## 管理员能力

- 配置 Bot Token、群组和命令。
- 管理花名册和换绑请求。
- 发送广播。
- 使用开发者 JS 沙箱扩展命令。

## 安全规则

- `TelegramMode` 控制 Bot 模式，不应阻止用户自助绑定。
- Bot Token、群组 ID 和内部密钥不得出现在公开文档、日志或普通接口响应中。
