# Twilight Legacy WebUI（Vite 原型）

该目录是 Twilight 早期前端原型工程（Vite + React），当前主开发不在此目录进行。

## 当前状态

- 用途：历史原型与实验代码保留
- 维护级别：低（不保证与当前后端接口完全一致）
- 推荐：新功能请在 `../`（Next.js 主前端）开发

## 本地运行（仅在需要调试原型时）

```bash
cd webui/webui
npm install
npm run dev
```
